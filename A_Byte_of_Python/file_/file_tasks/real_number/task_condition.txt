Дано файл, компоненти якого є дійсними числами. Скласти
підпрограму для обчислення:
a) суми компонент файлу;
b) кількості від’ємних компонент файлу;
c) останньої компоненти;
d) найбільшого із значень компонент файлу;
e) найменшого із значень компонент файлу з парними номерами;
f) суми найбільшого і найменшого із значень компонент;
j) різниці першої і останньої компоненти файлу;
h) кількості компонент файлу, які менші за середнє арифметичне усіх його компонент
